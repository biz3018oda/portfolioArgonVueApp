# 環境構築

## argon template

```
git clone https://github.com/creativetimofficial/vue-argon-design-system.git app
cd app
app % rm -rf .git
```

- git

```
cd app

app % git init
app % git branch -m main
app % git add .
app % git commit -m "Initial commit"
```

- npm

```
app % npm install
app % npm run serve
```
